# Audit LLM Assistant Playbook

A cognitive playbook for experienced smart contract auditors.

This is NOT an automated audit tool.
It does NOT replace auditor judgment.
It structures how auditors use LLMs during real audits.

## What this repository contains
- A single operational playbook used in real audits
- Structured workflows: exploration, hypothesis-driven analysis, validation, drafting, review
- Clear separation between core audit flow and supporting tools

## How to start
➡️ Read the playbook here: [Audit LLM Assistant Playbook](./Audit_LLM_Assistant_Playbook.md)

Start with:
1. "How to use this playbook"
2. "CORE AUDIT FLOW → BUILD LAYER"

## Who this is for
- Experienced smart contract auditors
- Security researchers already doing manual audits

## Who this is NOT for
- Beginners learning Solidity
- Automated vulnerability scanning
- “Run once and get bugs” workflows
